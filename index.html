Web Bomic Collect

Allows collection of Web Bomic client events via IPad or Android devices, that can be synchronised to any Web Bomic application.

Installing Web Bomic Collect

WB Collect is made of 3 parts that need to be installed separately.

  • A SQL Script that adds a pin number field to the users table (USRS_DAT) in Web Bomic.
  • A Web API interface that fetches and posts data into Web Bomic.

  • A IPad or Android App that allows the collection of the events.
